### 5.3.1 Video - Network Protocols
#### What is a Protocol
In both human and computer communications, protocols are the rules that govern how information is exchanged and understood.

The Human Analogy: > Just as humans use protocols like speaking a common language, deciding on formal vs. informal tones, greeting each other, or following dress codes, computers require structured rules to communicate effectively.

#### Computer Network Protocols
In computer networks, a single message is rarely sent using just one protocol. Instead, a message is made up of several protocols working together in a stack to ensure it is successfully delivered and understood.

#### Key Protocols Explained
|Protocol|Full Name|Primary Function|
|:--:|:--:|:--:|
|Ethernet|Ethernet Protocol|Governs physical communication from network interface card (NIC) to NIC within the same local network.|
|IP|Internet Protocol|Governs communication from the original source to the final destination. Routers use IP to guide packets along the way.|
|TCP|Transmission Control Protocol|Ensures reliable delivery. If packets arrive out of order, TCP rearranges them into the correct sequence.|
|HTTP|Hypertext Transfer Protocol|Governs the exchange or transfer of HTML (Hypertext Markup Language), which is used when requesting or downloading webpages.|
